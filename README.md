# 네이버 상세페이지 JPG 자동 제작기

Streamlit에서 상품 문구와 사진을 입력하면 네이버 스마트스토어에 바로 올릴 수 있는 긴 JPG 상세페이지를 만듭니다.

## 주요 변경사항

- HTML 다운로드 제거
- 완성된 긴 JPG를 앱에서 직접 생성하고 다운로드
- 기본 사진 11개 배치
- 추가 모델 착용컷 여러 장 업로드
- 추가 원단·봉제·단추 디테일컷 여러 장 업로드
- 추가 코디·기타 상세 이미지 여러 장 업로드
- 추가 사진은 세 구역 합계 최대 30장 반영
- 한글 JPG 렌더링을 위해 Noto CJK 폰트 자동 설치

## 고정 국내제작 이미지

`assets/domestic_production.jpg`가 모든 상세페이지의 09번 국내제작 구역에 자동으로 삽입됩니다.
앱 화면에서 해당 이미지를 다시 업로드할 필요가 없습니다. 다른 이미지로 바꾸려면 같은 파일명으로 교체하세요.


## 이번 버전 추가 사항

- 직접적인 연령 표현을 제거한 범용 기본 문구 적용
- 특정 디자인에 치우치지 않는 세일 포인트 기본값 적용
- 브랜드 스토리 3구역 추가
- 국내제작 고정 이미지 아래에 스토리와 단체 주문 안내 배치


## 상품 종류 프리셋 7종

- 블라우스형
- 원피스형
- 재킷형
- 티셔츠형
- 팬츠형
- 조끼형
- 아우터형

버튼 한 번으로 메인 문구, 세일 포인트, 소재 설명, 디테일 설명, 추천 착용 상황, 상품 정보 기본값이 상품군에 맞게 자동 적용됩니다.


## 추가 업데이트

- 상품 종류 프리셋 7종 지원: 블라우스형, 원피스형, 재킷형, 티셔츠형, 팬츠형, 조끼형, 아우터형
- 테마 색상 확장: 퓨어 화이트, 클래식 아이보리, 모던 블랙, 로즈 베이지, 세이지 그린, 딥 네이비, 테라코타 브라운, 라벤더 그레이
- 혼합형 상세페이지도 브랜드 스토리를 위쪽에 배치하도록 구성 개선
- 나염 패턴은 위치와 배열이 동일하지 않을 수 있다는 사전 안내 문구 기본 포함


## 문구 스타일 선택 기능

- 같은 상품군이라도 원하는 느낌에 맞춰 문구 스타일을 선택할 수 있습니다.
- 지원 스타일: 균형형 / 감성형 / 신뢰형 / 판매형
- 스타일 버튼을 누르면 현재 선택된 상품군 기준으로 메인 문구, 세일 포인트, 소재·디테일 설명, 하단 마무리 문구가 한 번에 바뀝니다.
- 자동 적용 후에도 모든 문구는 직접 수정할 수 있습니다.


## 상품명 기반 추천 기능

- 상품명을 입력하면 현재 선택한 상품군과 문구 스타일을 반영해 제목 3안과 메인설명 3안을 자동 생성합니다.
- 각 추천안은 개별 적용할 수 있으며 제목과 설명을 서로 다른 번호로 조합할 수 있습니다.
- 추천을 적용한 뒤에도 기본 문구 탭에서 자유롭게 수정할 수 있습니다.
- 국내원단 배지 문구는 `촉감과 조직감, 완성도를 고려한 소재`로 변경했습니다.


## 입력 화면 및 사이즈표 개선

- 기본 문구 / 세일 포인트 / 상품 정보 / 사진 업로드 탭을 크게 표시
- 실측 사이즈는 최대 8개 사이즈까지 행을 추가해 입력 가능
- 측정 항목명을 직접 바꿀 수 있어 상의, 원피스, 팬츠, 조끼, 아우터에 모두 대응
- 상품군 버튼을 누르면 측정 항목명도 상품 종류에 맞게 자동 변경
- 현재 문구, 테마, 상품정보, 실측 사이즈를 앱 기본값으로 저장하고 다시 불러오기 가능
- 기본값 JSON 다운로드 및 업로드 지원
- JPG 품질 기본값을 최대 95로 설정

앱 내부에 저장한 기본값은 Streamlit 서버 재배포나 재시작 시 초기화될 수 있으므로 JSON 파일도 함께 보관하는 것이 안전합니다. 업로드한 상품 사진은 기본값 저장 대상에 포함되지 않습니다.


## 상품정보 기본값 변경

- 상품정보에서 색상, 사이즈, 소재, 기타 항목 제거
- 제조국: 대한민국
- 제조자: 레드
- 세탁방법: 손세탁 및 단독 세탁 권장
- 실측 사이즈는 별도의 다중 사이즈 표에서 관리
