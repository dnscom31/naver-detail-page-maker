# 네이버 상세페이지 자동 제작기

GitHub와 Streamlit Community Cloud에서 바로 배포할 수 있는 상세페이지 제작 앱입니다.

## 주요 기능

- 상세페이지 테마 4종
  - 퓨어 화이트
  - 클래식 아이보리
  - 모던 블랙
  - 로즈 베이지
- 제목·본문 폰트 변경
- 페이지 폭·글자 크기·구역 간격 변경
- 판매 문구와 상품 정보 수정
- PC·휴대전화에서 상품 사진 업로드
- 실시간 미리보기
- 사진이 포함된 단일 HTML 다운로드
- 설정 JSON 저장 및 불러오기

## GitHub 저장소에 올릴 파일

아래 구조를 그대로 유지하세요.

```text
app.py
requirements.txt
README.md
.gitignore
.streamlit/
└── config.toml
```

## Streamlit Community Cloud 배포

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소 최상위에 업로드합니다.
3. Streamlit Community Cloud에 GitHub 계정으로 로그인합니다.
4. 새 앱 배포 메뉴를 선택합니다.
5. 저장소와 브랜치를 선택합니다.
6. 실행 파일 경로에 `app.py`를 입력합니다.
7. 배포 버튼을 누릅니다.

배포 후 GitHub의 코드를 수정하고 푸시하면 앱에도 반영됩니다.

## 사용 방법

1. 앱에서 테마와 폰트를 고릅니다.
2. 상품 문구와 상품 정보를 수정합니다.
3. 사진 업로드 탭에서 이미지를 넣습니다.
4. 화면 하단의 `상세페이지 생성하기`를 누릅니다.
5. 미리보기를 확인한 후 완성된 HTML을 다운로드합니다.

## 네이버 스마트스토어 등록

다운로드한 HTML은 사진이 파일 안에 포함된 단일 파일입니다.

스마트스토어에는 HTML 자체보다 긴 이미지로 등록하는 방식을 권장합니다.

Chrome에서 긴 이미지로 저장:

1. HTML 파일을 Chrome으로 엽니다.
2. `F12`를 누릅니다.
3. `Ctrl + Shift + P`를 누릅니다.
4. `Capture full size screenshot`을 검색합니다.
5. 생성된 PNG 이미지를 스마트스토어 상세설명에 등록합니다.

## 주의사항

- 업로드한 사진은 앱 실행 중 메모리에 사용됩니다.
- 새로고침하거나 세션이 종료되면 업로드 상태가 초기화될 수 있습니다.
- 공개 배포한 앱은 링크를 아는 사람이 접속할 수 있으므로 공개 범위를 확인하세요.
- 비밀번호 또는 관리자 로그인 기능은 기본 버전에 포함되어 있지 않습니다.
